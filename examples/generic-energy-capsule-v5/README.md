# Generic Energy Maintenance Capsule

Demonstrates that a domain without a built-in pack can still be discovered and
structured. Safety/compliance rules remain project-scoped and unverified until
the accountable owner and authoritative sources confirm them.

```powershell
py -3 scripts/validators/validate_project_domain_capsule.py examples/generic-energy-capsule-v5/project-domain-capsule.yaml
```
