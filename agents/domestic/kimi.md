# Kimi Adapter

Use Kimi's long-context capability for source inventory, not as a reason to
draft all artifacts in one pass. Build the full evidence/ID inventory, then
slice by module/flow and checkpoint decisions to prevent late-stage drift.
Freeze the approved ID index and
switch to small module slices. Save one Product Truth fragment and one PRD
section checkpoint at a time. Re-read only linked source IDs after interruption.
