## Submission Checklist

- [ ] README first screen explains the problem in English.
- [ ] Apache-2.0 license is present.
- [ ] Latest release tag exists (`v4.6.2` or newer).
- [ ] Changelog has a dated release section, not only `Unreleased`.
- [ ] Examples exist for CRM, traffic safety, education IT, and medical/hospital IT.
- [ ] Agent install paths are documented for clone, Skills CLI, and Claude Code.
- [ ] Contribution guide exists.
- [ ] Issue templates exist.
- [ ] The submitted description is one line, not promotional copy.
- [ ] The list is alphabetized if the target Awesome list requires it.
