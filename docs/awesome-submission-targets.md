## Submission checklist / 提交检查

- README first screen states the problem, audience, and evidence boundary.
- Installation and the 10-minute path use commands that exist in the repository.
- `status` output matches README maturity and evaluation claims.
- No client-specific project name or release-specific compatibility adapter is shipped.
- Domain fixture presence is not described as behavioral validation.
- Apache-2.0 license, security policy, contribution guide, and release tag are present.
- Description follows the target list's length, order, and category rules.
