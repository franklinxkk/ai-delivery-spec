## One-line positioning / 一句话定位

AI Delivery Spec is an open-source product delivery kernel: lightweight for
ToC idea-to-PRD work, and deep for governed ToB/ToG delivery. It turns rough
requests and existing systems into one Product Truth projected for humans,
coding agents, QA, customers, and operations.

AI Delivery Spec 是通用产品交付内核：ToC 想法到 PRD 可以轻量使用，ToB/ToG
复杂交付可以深度展开；所有视图由同一套 Product Truth 投影。

## Role hooks / 角色钩子

- PM: stop losing page effects, state boundaries, and acceptance between PRD and prototype.
- Architect: keep module, event, permission, and change contracts traceable.
- Coding agent user: block invented fields, states, permissions, and business rules.
- QA/customer delivery: trace every acceptance path back to evidence and decisions.
